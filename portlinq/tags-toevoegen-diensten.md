# Tags toevoegen aan je diensten (Data Service Provider)

Deze gids is voor **Charlie** - een data service provider die tags toevoegt aan geregistreerde API's en apps. Tags maken diensten vindbaar en consistent geclassificeerd in de catalogus.

## Huidige tags

Op dit moment zijn de volgende tags beschikbaar om aan diensten toe te voegen:

| Tag | Betekenis |
| --- | --- |
| `port` | Haven - geeft aan welke diensten bij een specifieke haven horen |
| `shorepower` | Walstroom - dienst voor walstroomvoorziening |
| `visit` | Scheepsbezoek - dienst rond het bezoek van een schip aan de haven |
| `vessel` | Vaartuig - hiermee kun je filteren welke organisatie een vessel-dienst aanbiedt |

> ℹ️ Je kunt alleen tags uit deze lijst kiezen. Mis je een tag die je nodig hebt? Neem contact op met de PortlinQ-beheerder - zie [Tags beheren](tags-beheer.md).

## Werkwijze

1. Log in op het [Self-Service Portal](https://portlinq-preview.poort8.nl/portal)
2. Open de detailpagina van je API of app
3. Selecteer een of meer tags uit de beschikbare lijst
4. Sla de toewijzing op

Tags zijn zichtbaar op de detailpagina van je API of app, zodat consumers meteen zien waar de dienst voor bedoeld is.

## Tags weer verwijderen

Klopt een tag niet meer? Ga terug naar de detailpagina van je API of app en verwijder de tag. Dit heeft geen invloed op bestaande toegangsrechten van consumers - het verandert alleen de classificatie in de catalogus.

Vragen? Neem contact op met Poort8 via **<hello@poort8.nl>**.
